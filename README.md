# 🚀 Unified LPG Booking System

A Java-based **Console Application** that simulates an LPG gas booking platform. The system enables customers to register, log in, book LPG cylinders, manage bookings, make payments, and allows administrators to manage cylinder stock and users. The project is built using **Core Java** with Object-Oriented Programming principles and demonstrates a modular software architecture.

---

## 📌 Features

### 👤 Customer Module
- Customer Registration
- Secure Login
- View Profile
- Book LPG Cylinder
- Choose Cylinder Type
  - Domestic
  - Commercial
- Select LPG Brand
  - Indane
  - Bharat Gas
  - HP Gas
- Online Payment
- Cash on Delivery
- Booking History
- Subsidy Calculation

### 👨‍💼 Admin Module
- Admin Login
- Manage Customers
- View All Bookings
- Manage Cylinder Inventory
- Update Cylinder Stock
- Monitor Booking Details

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Kishore-15072004/UnifiedLPGBookingSystem.git
```

2. Open the project in Eclipse IDE.

3. Navigate to:

```
src/com/lpg/main/Main.java
```

4. Run the `Main.java` file.

5. Follow the console menu to use the application.
